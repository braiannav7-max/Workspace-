# CTA — `listo`

**Categoría:** cta · **Fuente:** HyperUI · **Licencia:** MIT

## Qué es
Bloques de cierre (call-to-action) para empujar al usuario a la acción: registrarse, comprar o pedir una demo. Estética limpia tipo Linear/Stripe con acento indigo, tipografía Inter y animación AOS fade-up.

## Cuándo usar cada variante
- **`cta-banner.html`** — caja indigo sólida dentro del contenido. Ideal a mitad de página o entre secciones, sin robar protagonismo.
- **`cta-fondo.html`** — sección full-width con gradiente vistoso y overlay. Más "wow"; pensada para el cierre final de una landing.

## Qué personalizar
- **Título y subtítulo:** reemplazá los textos del `<h2>` y `<p>`.
- **Botones:** cambiá el texto y el `href` del CTA primario y secundario.
- **Color / gradiente:** ajustá `bg-indigo-600` (banner) o `from-indigo-600 via-purple-600 to-fuchsia-600` (fondo).

> Cada `.html` es autocontenido: se abre solo en el navegador, sin build ni dependencias locales.
