# Features — `listo`

**Categoría:** Features · **Fuente:** HyperUI · **Licencia:** MIT

## Qué es
Bloques de sección "features" copy-paste para presentar las funcionalidades o beneficios de un producto. Estética limpia tipo Linear/Vercel/Stripe, acento indigo-600, tipografía Inter y animación de entrada con AOS (stagger).

## Cuándo usar cada variante
- **features-grid.html** — Grid de 6 features con ícono en cuadrado redondeado. Ideal para listar muchas funcionalidades de forma compacta.
- **features-iconos.html** — Grid de 4 features con ícono grande circular y todo centrado. Más aireado, para destacar pocos pilares clave.
- **features-alternado.html** — Filas alternadas imagen/texto con mini-listado de checks. Ideal para explicar funcionalidades en detalle con apoyo visual.

## Qué personalizar
- Títulos y subtítulos de cada sección y feature.
- Textos descriptivos y los ítems del listado de checks.
- Iconos SVG inline (reemplazá el `path` por cualquier heroicon stroke).
- Color de acento: buscá y reemplazá `indigo-600` / `indigo-50`.
- Imágenes: cambiá las URLs de `placehold.co` por las reales.

> Cada `.html` es autocontenido: se abre solo en el navegador sin build ni dependencias locales.
