# Footer — `listo`

**Categoría:** Footer · **Fuente:** HyperUI · **Licencia:** MIT

## Qué es
Bloques de pie de página oscuros (bg-slate-900) con acento indigo-600 e Inter. Tres variantes copy-paste, cada una en un `.html` autocontenido y abrible solo.

## Cuándo usar cada variante
- **footer-simple** → landings o sitios chicos: logo, links horizontales, redes y copyright.
- **footer-links** → sitios con varias secciones: 4 columnas (Producto, Empresa, Recursos, Legal) + columna de marca.
- **footer-newsletter** → cuando querés captar emails: igual a links + bloque de suscripción con JS vanilla.

## Qué personalizar
- **Logo:** el cuadrado "M" y el texto "Marca".
- **Links:** los `<a href="#">` de cada columna/nav.
- **Redes:** los SVG inline (Twitter, Instagram, LinkedIn).
- **Color:** acento `indigo-600` y fondo `slate-900`.

> Nota: archivos autocontenidos (Tailwind CDN + AOS por CDN). Para producción, copiá solo el `<footer>` a tu proyecto.
