# Hero — `listo`

**Categoría:** Hero · **Fuente:** HyperUI · **Licencia:** MIT

## Qué es
Sección principal (hero) que encabeza una landing: título, subtítulo y llamada a la acción. Seis variantes copy-paste (3 base + 3 premium animadas), cada una autocontenida.

## Cuándo usar cada variante

**Base (limpias y rápidas):**
- **hero-fullscreen.html** — Ocupa toda la pantalla con gradiente sutil y dos CTAs. Ideal para landings de impacto donde el hero es protagonista.
- **hero-split.html** — Dos columnas (texto + imagen). Perfecto cuando querés mostrar el producto al lado del mensaje. Se apila en mobile.
- **hero-minimal.html** — Muy limpio, centrado, con badge y un solo CTA. Para marcas que priorizan el espacio en blanco y un mensaje directo.

**Premium (con animación, para impresionar al cliente):**
- **hero-aurora.html** — Fondo oscuro con "aurora" de gradientes que se mueven en loop (CSS). El más "wow". Para marcas modernas/tech.
- **hero-typing.html** — Título con efecto máquina de escribir que cicla palabras (JS). Bueno para agencias/servicios ("Construimos webs que venden / apps a medida / ...").
- **hero-flotante.html** — Tarjetas y figuras que flotan con parallax al mover el mouse (CSS + JS). Da profundidad y sensación de producto vivo.

## Qué personalizar
- **Título** y **subtítulo**: cambiá el texto principal.
- **Color de acento**: reemplazá `indigo-600` (y `indigo-700` en hover) por tu color.
- **Imagen**: en la variante split, cambiá el `src` del `<img>` placeholder.

> Nota: los tres `.html` son autocontenidos y se abren directamente en el navegador.
