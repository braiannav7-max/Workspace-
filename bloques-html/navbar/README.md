# Navbar — `listo`

**Categoría:** ui · **Fuente:** HyperUI (patrones de header) · **Licencia:** MIT

## Qué es
Barras de navegación superiores para landings, en 3 variantes.

## Cuándo usar cada variante
- **`navbar-sticky.html`** → se queda fija arriba con blur al scrollear. Uso general.
- **`navbar-transparente.html`** → transparente sobre un hero con imagen/gradiente; se vuelve sólida al bajar.
- **`navbar-mobile.html`** → incluye botón hamburguesa con menú desplegable. Base recomendada para responsive.

## Qué personalizar
- Logo: la "M" y el texto "Marca".
- Links del menú (`Inicio`, `Servicios`, etc.).
- Color principal: reemplazar `indigo-600` por el color de la marca.

## Notas
Autocontenidas (Tailwind por CDN). El menú mobile y el efecto transparente usan JS vanilla inline.
