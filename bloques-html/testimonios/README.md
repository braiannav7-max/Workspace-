# Testimonios — `listo`

**Categoría:** Testimonios · **Fuente:** HyperUI · **Licencia:** MIT

## Qué es
Bloque de prueba social en 3 variantes copy-paste, autocontenidas (Tailwind CDN + Inter + AOS). Cada `.html` se abre solo en el navegador, sin build.

## Cuándo usar cada variante
- **testimonios-cards.html** → grid de 6 reseñas con estrellas. Ideal para mostrar volumen y variedad de opiniones.
- **testimonios-carrusel.html** → un testimonio destacado a la vez con prev/next y dots. Para resaltar pocas citas potentes.
- **testimonios-logos.html** → banda de logos de clientes + métricas. Para transmitir confianza y escala.

## Qué personalizar
- **Quotes y nombres/cargos:** editá el texto de cada card o el array `testimonios` (carrusel).
- **Avatares:** cambiá el `N` en `i.pravatar.cc/80?img=N` (1..70) o usá tus propias imágenes.
- **Logos:** reemplazá los nombres/SVGs por los de tus clientes reales.
- **Color de acento:** buscá y reemplazá `indigo-600` por tu color de marca.

> Nota: el carrusel usa **JS vanilla inline** (sin librerías) para prev/next, dots y transición de opacidad.
